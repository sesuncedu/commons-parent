              Apache Commons Parent 36
                  RELEASE NOTES

The Apache Commons Parent team is pleased to announce the release of Apache Commons Parent 36

The Apache Commons Parent POM provides common settings for all Apache Commons components.


Changes in this version include:

Update plugins, fix release-notes profile.


Fixed Bugs:
 o Add announcementDirectory to release-notes profile.

Changes:
 o Update maven-jxr-plugin 2.4 to 2.5
 o Update maven-surefire-plugin(s) 2.17 to 2.18
 o Update maven-assembly-plugin 2.5 to 2.5.1
 o Update maven-clean-plugin 2.6 to 2.6.1
 o Update commons.wagon-ssh.version 2.6 to 2.8


Historical list of changes: http://commons.apache.org/changes-report.html

patches, or suggestions for improvement, see the Apache Apache Commons Parent website:
